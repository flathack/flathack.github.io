# flathack.github.io

Static GitHub Pages site for project information, status tracking, wiki pages, and program documentation.

## Structure

- `index.html` landing page
- `assets/css/style.css` shared design
- `projects/` project overview pages
- `status/` development status pages
- `wiki/` wiki index and articles
- `docs/` program documentation
- `changelog/` release and progress notes
- `about/` site and author information

## Publishing

This repository is intended for GitHub Pages using the `main` branch.

## Editing workflow

1. Update content directly in the relevant HTML page.
2. Commit changes.
3. Push to `main`.
